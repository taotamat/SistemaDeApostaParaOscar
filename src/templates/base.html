SOBRE

O sistema de apostas para o Oscar é um software responsável pela divulgação das listas por categorias de diversos segmentos selecionados à premiação do Oscar, como também os registros e gerenciamentos de apostas para premiação. O principal objetivo do sistema é registrar e fazer o controle das apostas entre os usuários do sistema, além de cadastrar as categorias e divulgar a lista selecionada da premiação. O sistema possui como principais públicos alvos: os usuários (apostadores e administradores do sistema).
